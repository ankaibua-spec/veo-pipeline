# VEO AI Studio

Web UI for **VEO Pipeline Pro** — generated by [Stitch](https://stitch.withgoogle.com), refined per [DESIGN.md](https://github.com/ankaibua-spec/veo-pipeline/blob/main/DESIGN.md).

**Author:** Truong Hoa · Zalo `0345431884`

## Stack

- Vite 6 + React 19 + TypeScript
- Tailwind CSS v4 (`@theme` API)
- Motion (animations)
- Lucide icons
- `@google/genai` (Gemini SDK ready)

## Quick start

```bash
npm install
cp .env.example .env  # add GEMINI_API_KEY
npm run dev
```

Open http://localhost:3000

## Build + Deploy

```bash
npm run build       # static dist/
npm run preview     # local preview
```

Deploy to Vercel / Netlify / Cloudflare Pages. Static export — no server required.

## Brand

Strict adherence to desktop tool's `DESIGN.md`:
- Primary `#0078D4` · Accent `#FF6B35`
- Surfaces `#1F1F1F` / `#252525` / `#2B2B2B` / `#3A3A3A`
- Inter font (web fallback for Segoe UI Variable)
- Dark mode only · No gradients on content
